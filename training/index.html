Training

Equip your staff with the expertise, and confidence to work with open data.

  • Open Data in One Day

    This one day course is a short introduction suitable for anyone interested in open data, and a great fit for teams from diverse roles across organisations who are considering adopting open data for the first time or trying to understand how it would be useful to them. The course covers the “what”, “why” and how” of open data and as such is suitable for anyone. This course is delivered face-to-face and is available for up to 12 delegates at a suitable location of your choice.

    £2,500
  • Open Data Management - 5 days

    This course is designed for those specializing in open data, whether as policy experts, open data program managers and similar roles in government, civil service, and other organisations.  The course is non-technical in nature and deals with the strategic considerations of managing an Open Data programme within an organisation. Such activities typically include implementing an Open Data strategy, designing and launching an Open Data portal, coordinating publication processes, preparing data for publication, and fostering data re-use.

Technical modules

We provide a range of technical modules delivered face-to-face at a suitable location of your choice. Please see below our frequently requested modules. We are also able to offer custom modules. Please allow for additional preparation time for custom modules.

  • Introduction to CKAN

    Time Required: 2-4 hours
    Audience: Non-technical staff and developers
    Prerequisites: None

    By the end of this course delegates will know what CKAN is, understand CKAN’s key concepts (for example: the concepts of dataset and resource), know how to add data to CKAN and use its search, data management and personalization features, data visualization capabilities, API and extensions.

  • Geospatial features

    Time Required: 4 hours
    Audience: Developers
    Prerequisites: Introduction to CKAN module

    By the end of this course delegates will know how to deploy and use CKAN’s geospatial search, harvesting, CSW and preview features provided by the ckanext-spatial plugin.

  • The CKAN API

    Time Required: 2-4 hours
    Audience: Developers
    Prerequisites: CKAN install, Python

    By the end of this course delegates will know how to write Python apps and scripts that interact with CKAN sites via the CKAN API.

  • Harvesting Data

    Time Required: 4 hours
    Audience: Developers
    Prerequisites: Introduction to CKAN module

    By the end of this course delegates will know how to deploy and use CKAN’s data harvesting features, provided by the ckanext-harvest extension.

  • Writing CKAN plugins

    Time Required: 4 hours minimum
    Audience: Back-end (Python) Developers
    Prerequisites: CKAN install, Python, CKAN API

    By the end of this course delegates will know how to customize and extend CKAN’s functionality by writing your own CKAN plugins.

  • Integrating CKAN with your Content Management System*

    Time Required: 2 hours
    Audience: Developers
    Prerequisites: CKAN API, Writing CKAN plugins

    By the end of this course delegates will know how to integrate CKAN with a content management system like Drupal or WordPress.

    * Note: Please allow an extra day for detailed consultation to find out the exact requirements of your system for this course.
